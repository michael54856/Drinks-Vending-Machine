# Drinks-Vending-Machine
嵌入式期末Project - 自動飲料販賣機

## 介紹

*本次的專案將利用Arduino製作一臺飲料自動販賣機。
*有別於利用硬幣大小來做硬幣辨識，本次專案將利用顏色來識別硬幣，並根據該顏色來提供飲料給使用者。
*該機器可運用在公司、住家、或是派對聚會上。
